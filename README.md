# VN1000 teaching slides

Published (GitHub Pages) copy of the teaching slide deck for the **VN1000 / Linux NIC Simulation**
project. The main project repository is private; this repo mirrors only the rendered slide deck and
its diagrams so the deck can be viewed in a browser.

- **View the slides:** https://oopsdev.github.io/vn1000-slides/
- `index.html` - the deck (vanilla JS: arrows / buttons / `F` fullscreen / page jump / `P` PDF).
- `diagrams/` - the SVG figures it embeds.
